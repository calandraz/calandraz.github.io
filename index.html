Pomodoro Timer

As part of the Free Code Camp curriculum, this pomodoro is a time management site that allows the user to choose how long they wish to work and how long they wish to engage in breaks. Pomodoro Timer was built with HTML5, Bootstrap, Flipclock.js, and JavaScript.

  • Client: Free Code Camp
  • Date: February 2016
  • Service: Web Development

Twitch.TV

A Free Code Camp project built using HTML5, CSS3, Javascript/jQuery, and Twitter Bootstrap. This project utilizes the Twitch.TV api to gather JSON formatted information on a predetermined set of streamers. This webpage lists the streamers, their status, and what game they are currently streaming.

  • Client: Free Code Camp
  • Date: February 2016
  • Service: Web Development

Tic-Tac-Toe

A Free Code Camp project of Tic-Tac-Toe, this version of the game was built using the min-max algorithm, making it a challenge to win against the computer. Tic-Tac-Toe was developed using HTML5/BootStrap, CSS3, and JavaScript.

  • Client: Free Code Camp
  • Date: April 2016
  • Service: Web Development

Simon Says

Based on the popular 1970's Hasbro game, Simon Says is another project built for the Free Code Camp curriculum. Equipped with both a normal and strict mode, this game slightly increases in difficulty as the user proceeds. Simon Says was developed using HTML5, Bootstrap, CSS3, and JavaScript.

  • Client: Free Code Camp
  • Date: April 2016
  • Service: Web Development

Markdown Editor

This Markdown Editor is the first project built with the popular library, React.js. Using Marked library, the user is able to input GitHub-flavored markdown and see an output of the markdown as it is typed.This project was developed using React.js and Sass.

  • Client: Free Code Camp
  • Date: June 2016
  • Service: Web Development

TimeStamp Microservice

Developed for Free Code Camp's curriculum, the Timestamp Microservice is the first backend project. This project allows the user to input a string parameter and output a unix timestamp and a natural language data. This project was developed with HTML5/Bootstrap, CSS3, Node.js/Express.js and deployed to Heroku.

  • Client: Free Code Camp
  • Date: May 2016
  • Service: Web Development
